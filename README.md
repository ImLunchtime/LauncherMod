# Launcher IM Mod
A remix of the original Launcher. Removed lots of board support and lots of functions.
Currently based on Launcher 2.6.5, so it contains the *currently* latest features..

### Removed Functions
 - WebUI
 - SD-USB Interface

### Supported Devices
 - M5Stack Cardputer/ADV
 - M5Stack Tab5
 - M5Stack CoreS3
